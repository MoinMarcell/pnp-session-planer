## PnP Session Planer

### <u>Description</u>
This project is a session planer for Pen and Paper events. It is a web application that allows users to create and manage events and sessions for Pen and Paper events.

### <u>Backend</u>

#### Technologies:
- Java 21
- Spring Boot 3.2.3
- Maven 4.0.0
